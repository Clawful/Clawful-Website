# Removewarn

#### Description

 _Removes the selected warn from user_

| _Usage_ |
| :---: |
| cw!ban &lt;User ID/Mention&gt; &lt;Warn Number&gt; |

#### Arguments

| Parameters | Type | Optional |
| :---: | :---: | :---: |
| User ID/Mention | String or number | ❌ |
| Reason | String, reason for the ban | ❌ |
